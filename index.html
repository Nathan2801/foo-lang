Hello

Foo is a fictional programming language aiming to be the best one ever made.

To achive a optimal performance there is no question about Foo being a compiled language.

The memory managment is still under consideration since is a delicated topic which strongly defines how a programming language is percieved and used.

Types

Foo is a strongly typed language, here are the basic types we got in the language.

  • i32
  • i64
  • f32
  • f64
  • bool
  • list
  • string

Variables

For defining variables we want something simple, that's why was choosed the format name : type = value for variables and name : type : value for constants.

Here the example of how define a variable.

foo := 1 bar : i32 = 1

And here the example of how define a constant.

foo :: 1 bar : i32 : 1

Functions

Functions tries to follow the same syntax rules as constants, except we not require the second : due to readability.

Here is how to define a function without arguments and return value.

do_nothing : () {   return; }

And here is how to define function with arguments and a return value.

sum : (a i32, b i32) i32 {   return a + b; }

Conditions

There are no much to talk about conditions, they are C-like conditions without parenthesis.

condition :: true if condition {   // ... } condition :: true if condition {   // ... } else if {   // ... } else {   // ... }

We should as well have support for pattern match conditions, and we do so by using the case keyword.

condition :: true case condition {   true { //* ... *// }   false { //* ... *// } }
